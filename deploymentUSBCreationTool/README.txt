USB Deployment Creator - Jake Dixon, Eckhold Technologies
----------------------------------------------------------
1. ensure the contents of this directory are copied to a hdd before running as files generated by this program
   can be larger than the fat32 filezie limit before processing
2. copy all .wim or .esd files into the images folder before running
3. ensure you have a fat32 formatted usb drive to store the contents of the output directory after running the tool
4. copy the contents of the output directory to the usb drive
5. running the tool will delete any old output folders.
